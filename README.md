# dotfiles

MacOS only

## Tools

- GNU stow
- Homebrew bundle

# Installation

Run install

```
curl -o - https://raw.githubusercontent.com/nuovotaka/dotfiles/main/install | sh
```

Start Yabai and skhd

```
brew services start yabai
brew services start skhd
```

```
npm i -g typescript-language-server
```
