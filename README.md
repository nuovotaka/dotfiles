# Installation

Run install

```
curl -o - https://raw.githubusercontent.com/nuovotaka/dotfiles/main/install | sh
```

Install kitty

```
curl -L https://sw.kovidgoyal.net/kitty/installer.sh | sh /dev/stdin
```
